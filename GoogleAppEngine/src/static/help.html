Help

Type LaTeX expression in the input box and press UnicodeIt. The result can be copied into most programs including presentation software like PowerPoint and Keynote, e-mail clients like Outlook and Mail and even apps on smart phones. If you need a greek letter, an arrow or some other symbol in your presentation/email/chat/facebook, UnicodeIt creates it for you.

For more information, please visit www.svenkreiss.com/UnicodeIt.
